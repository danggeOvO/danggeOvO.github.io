Python语言的特点

解释型语言

解释型语言需要解释器，解释型过程就是将源代码逐条转换成目标代码同时逐条运行的过程。

解释型语言依赖于解释器，需要相应平台提供，所以可以跨多个平台进行；但是由于是逐行解释，程序执行效率低。
